# \# Nespad

# Nespad is a NES controller layout for keyboard input, similar to \[GamePad Viewer](https://gamepadviewer.com/).

# 

# \## Installation

# \[Download](https://github.com/itzmeunreal/Nespad/releases/tag/v1.0) and extract the latest release ZIP and run `Nespad.exe`.

# 

# \## Usage

# Press `Esc` to show or hide the menu.

# 

\### File
- \*\*Save and load layouts:\*\* save your current key bindings and skin as a named layout to reuse later.
- \*\*Reload:\*\* reloads the controller view.
===

# \- \*\*Quit:\*\* closes the app.

# 

# \### Settings

# \- \*\*Input:\*\* assign keyboard keys to each controller button.

# \- \*\*Skins:\*\* change the color of the controller.
